# 捕食/保护系统开发

## 目标

先搭建捕食与保护的基础逻辑框架，不接入任何具体动物能力。

本阶段只解决通用流程：

1. 动物发起捕食请求。
2. 系统根据规则寻找猎物目标。
3. 系统检查目标是否受到保护。
4. 未被保护的目标被捕食移除。
5. 返回捕食结果。
6. 留出事件扩展点，供后续动物能力监听。

本阶段暂不开发：

- 老虎、狼、狐狸等具体捕食能力。
- 狗、大鹅、长颈鹿等具体保护能力。
- 捕食奖励金币、化石、道具或掉落物。
- 捕食计数、成长、进化。
- 捕食动画、UI 提示、音效。
- 动物资产批量创建或修改。

## 核心概念

### PreyContext

一次捕食请求的上下文。

建议包含：

- 捕食者动物。
- 捕食目标规则。
- 捕食数量限制。
- 捕食来源能力 ID，可选。

### PreyTargetRule

猎物筛选规则。

建议包含：

- `impactType`：捕食范围，如 `Self`、`Adjacent`、`Row`、`Field`。
- `targetAnimalIds`：允许捕食的动物 ID。
- `targetFamilies`：允许捕食的动物家族。
- `excludeAnimalIds`：排除动物 ID。
- `excludeFamilies`：排除动物家族。
- `targetCount`：最多捕食目标数量。
- `randomPick`：是否从候选目标中随机选择。

### PreyResult

一次捕食的执行结果。

建议包含：

- 是否成功。
- 捕食者。
- 候选目标列表。
- 被保护目标列表。
- 实际被移除目标列表。
- 失败原因。

### ProtectionContext

一次保护判定的上下文。

建议包含：

- 捕食者。
- 被捕食目标。
- 候选保护者。
- 捕食请求来源。

### ProtectionRule

保护规则。

建议包含：

- `scope`：保护范围，如 `Self`、`Adjacent`、`Field`。
- `protectedAnimalIds`：可保护的动物 ID。
- `protectedFamilies`：可保护的动物家族。
- `againstPredatorIds`：可阻挡的捕食者 ID。
- `againstPredatorFamilies`：可阻挡的捕食者家族。

### ProtectionResult

保护判定结果。

建议包含：

- 是否保护成功。
- 保护者。
- 被保护目标。
- 保护原因。

## 基础流程

```text
TryPrey(context)
-> PreyTargetResolver 解析候选猎物
-> 对每个候选猎物进行保护检查
-> 已被保护的目标写入 protectedTargets
-> 未被保护的目标写入 removedTargets
-> 通过捕食移除入口移除目标
-> 返回 PreyResult
```

## Ranch 层入口

捕食系统需要 Ranch 层提供统一入口。以下文件不属于动物/能力专属范围，正式修改前需要先确认。

### RanchManager

建议新增：

```csharp
public PreyResult TryPrey(PreyContext context)
```

用途：

- 作为能力系统调用捕食的唯一入口。
- 避免具体能力脚本直接移除动物。
- 统一处理保护、移除、事件通知。

### RanchAnimalService

建议新增：

```csharp
public bool TryRemoveAnimalByPrey(Animal predator, Animal target)
```

用途：

- 区分普通移除和捕食移除。
- 为后续 `OnAnimalPreyed`、`OnPreySuccess` 等事件留下语义。

## 事件扩展点

第一版可以只定义同步事件，不接具体动物能力。

建议事件：

- `OnPreyAttempt`
- `OnPreyProtected`
- `OnPreySuccess`
- `OnPreyFailed`
- `OnAnimalPreyed`
- `OnAnimalRemoved`

后续用途：

- 奶牛监听相邻牛类被捕食。
- 火鸡监听自身被捕食。
- 鬣狗监听相邻动物被捕食。
- 狗、大鹅、长颈鹿统计保护次数。

## 第一版验收标准

不接入具体动物能力时，第一版只需要验证框架行为：

1. 可以构造一个捕食请求。
2. 可以按范围解析候选猎物。
3. 可以按动物 ID 或家族筛选目标。
4. 可以返回无目标的失败结果。
5. 可以返回被保护的结果。
6. 可以通过捕食移除入口移除未保护目标。
7. 捕食移除不破坏现有普通移除逻辑。

## 后续接入顺序

基础框架完成后，再按以下顺序接入具体能力：

1. 食肉动物基础捕食：老虎、狼、狐狸、雌狮。
2. 基础保护：狗、大鹅。
3. 被捕食触发：火鸡、水牛扩展。
4. 相邻捕食监听：奶牛、鬣狗。
5. 复杂捕食：剑齿虎、鹈鹕、雪豹、鳄鱼。
